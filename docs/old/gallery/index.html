mrvプレイヤー

「most-real-video」最もリアルな映像——という、そのあまりに傲慢な冠称とは裏腹に、中身は悪戯心に満ちた映像形式です。わたしが「或いは夜」の名義を手に入れたころ、初めてできたネット上の友人に見せた記憶があります。ちょうど、こんな木枯らしの吹く季節だったでしょうか。

アナモルフォーシスGen

「アナモルフォーシス」というトリックアートの一種があります。正面から見ればただの醜い染みに過ぎないものが、ある一点、斜め底からの視線を得た瞬間にだけ、鮮烈な「意味」へと結実する。 私たちの記憶も案外そうではないのではないかと思うのですが——....ともかく、あれは文化祭の折に急ピッチで（AIを使って）仕立てた習作でした。あの狂騒の中で、私はトリックアートを編み出すことになった。このレポジトリは、それを完遂するための、私だけの視点なのです。

Anti_CORS

CORS、というブラウザ上のセキュリティシステムがあります。これは、ページが読み込めるものを同一オリジンのもののみに制限するシステムです。本来、インターネットに潜む攻撃者から私たちを守ってくれるものなのですが、当時の私はこれがどうも好きではなかったようで。前述のアナモルフォーシスGenを作ったころ、私はとあるパラドックスに陥っていました。「学校のPCってなんかGithub Pages表示してくれないんだよな....かといってGithubから落としてもローカルサーバー立てられないしな....」....というわけで作りました、プロジェクトそのものを拡張機能でラップしてchrome-extension:という同一オリジンを手に入れるというハックを。 これも急ピッチで作ったものなのでCSSあたりがAIっぽいですね。

hello world

HelloWorldの出力に700行使いました。具体的に言うと 不変性とフェイルファストを軸とした、極めて防御的な疎結合アーキテクチャであり、DIコンテナによるライフサイクル管理、FSMによる厳密な状態遷移制御、およびCQRS/イベントソーシングを用いた非同期メッセージングを統合することで、単一責任の原則を徹底し、実行環境の差異に依存しない「予測可能な動作」を数学的・構造的に保証する。 ものです。
