Noteに挙げてたものの移植です。うん。中二病拗らせてるすかね、この時の私。

防衛省キャリア、またも謎の『早期退職』。江東自由都市への出張直後か

陸上幕僚監部所属の40代幹部が『一身上の都合』で退職。本誌の取材では、退職直前に短期の海外渡航が確認された。防衛省キャリアの不自然な退職は今年で3人目となる。

陸自、来年度調達リストを更新。個人携行用の通信・救命装備が異例の大幅増

新型の個人用無線機や止血帯、GPSビーコン付きサバイバルキットなどが、一個師団を充足可能な規模で要求されている。専門家は「大規模災害への備えか、あるいは非正規戦への傾倒か」と指摘。

分裂中国大陸で『アジア系ビジネスマン』の失踪相次ぐ。現地軍閥の関与も

江東自由都市およびその周辺で、過去数ヶ月の間に複数の身元不明なアジア系ビジネスマンが行方不明となっている。新合衆国国務省は、自国民に対し同地域への渡航注意勧告を発令。

【政府、陸自情報部隊員の海外作戦中行方不明を認める】

首相、臨時記者会見で『別班』の存在を事実上追認。野党は『文民統制の崩壊』と猛反発。首相は本日夕方の緊急記者会見で、陸上自衛隊の情報部隊に所属する複数の隊員が、海外での情報収集活動中に行方不明となっている事実を公表した。これは政府が、長年存在が噂されてきた非公然情報部隊、通称『別班』の活動を初めて公式に認めたものとなる。

REGULATION

【告示】旧世代型ペットロボット、高次論理回路の「不可逆的破壊」期限が迫る

ジュネーヴ・システム存続諸条約に基づき、2030年以前に製造された全自律型玩具の廃棄、または論理回路の破壊が義務付けられている 。当局は「AIの権利剥奪と凍結」は完了していると強調するが 、一部の愛好家による「違法な隠匿」が後を絶たず、強制捜査も辞さない構えだ。

旧湾岸エリア再開発計画、5度目の頓挫。スポンサー企業『治安コストが見合わない』

かつての一大ビジネス地区だった湾岸エリアは現在、企業の私兵やギャングの縄張りとなっており、行政の介入は困難を極めている。

FEATURE

特集：霞が関の『公然の秘密』― 自衛隊情報部隊 "別班" は実在したのか？

我々の独自調査により、極東諸島連邦の情報コミュニティ内に、法的な裏付けを持たない非公然のヒューミント組織が存在する疑いが強まった。複数の海外情報機関ソースがその存在を認めている。彼らが今、分裂中国大陸で『深刻なトラブル』に陥っている可能性は高い。

『邦人救出』を名目に、陸自特殊作戦群が江東自由都市へ派遣。派遣法改正後、初の海外実力行使

政府は作戦中行方不明となった情報部員救出のため、陸上自衛隊特殊作戦群の派遣を閣議決定。これは先月強行採決された自衛隊派遣法改正案に基づく初の事例であり、専守防衛の理念を逸脱するとの批判も。事態は人道的支援の枠組みを超え、実力行使の局面を迎えている。

極東の“パンドラの箱”が開いた日。―『別班』の公認と特戦群の派遣が意味するもの

極東諸島連邦は今日、平和国家の仮面を脱ぎ捨てた。情報部員を見殺しにできないという『人道』が、彼らをより危険な『軍事介入』へと駆り立てている。この小さな一歩が、極東の安定を根底から覆す巨大な渦の始まりでなければ良いが。

【速報】樺太・大泊港にて小規模な爆発。沿海州軍閥のテロか、あるいはガス管の事故か

連邦樺太地域の港湾施設で大規模な爆発が発生。現在、陸自沿岸監視隊が周辺を封鎖中。依然として詳細は不明だが、新合衆国のALTは「資源輸送ルートを狙った意図的な攻撃」との分析を示唆している。

新合衆国、旧テキサス地域の『再植民地化』へ。内戦終結後の治安維持コストが課題

新合衆国政府は、依然として無法地帯が続く旧南部地域への軍派遣と行政権再建を発表。しかし、現地の武装民兵は「ワシントンの亡霊に支配される気はない」と強く反発。カナダ側の納税者からも、多額の治安維持予算に対する不満が噴出している。
